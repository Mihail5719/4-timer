# 🎄 Новогодний Таймер Обратного Отсчета

Интерактивный веб-сайт с таймером обратного отсчета до Нового года, поддерживающий несколько городов, смену темы (день/ночь) и праздничные эффекты.

## ✨ Основные возможности

- 🌍 **Мультигородской режим:** Переключение между Лиссабоном (UTC+0), Москвой (UTC+3) и Нью-Йорком (UTC-5). Таймер автоматически рассчитывает точное время до полуночи в выбранном часовом поясе.
- 🌅 **Смена темы (День/Ночь):** Динамическая смена фоновых изображений (слайд-шоу) в зависимости от выбранного режима.
- 💾 **Сохранение настроек:** Выбранный город, тема и состояние звука сохраняются в `localStorage` и восстанавливаются при перезагрузке страницы.
- 🎊 **Праздничные эффекты:** 
  - Салют из конфетти ровно в 00:00:00.
  - Бой курантов начинается за 41 секунду до наступления Нового года.
- 📱 **Полная адаптивность:** Корректное отображение и оптимизированные изображения как для десктопов, так и для мобильных устройств.
- 🎵 **Управление звуком:** Кнопка включения/выключения фоновой музыки и курантов с учетом политики автовоспроизведения (Autoplay Policy) современных браузеров.

## 🛠️ Технологии

- **HTML5** (семантическая разметка)
- **CSS3** (Flexbox, адаптивный дизайн, прозрачность, анимации)
- **JavaScript (ES6+)** (работа с датой и временем, `localStorage`, манипуляции с DOM, обработка событий)
- **canvas-confetti** (библиотека для анимации конфетти)
