#  Таймер обратного отсчёта до Нового года

Веб-приложение, которое в реальном времени показывает, сколько времени осталось до наступления Нового года. Проект демонстрирует работу с датами, асинхронностью и DOM-манипуляциями в чистом JavaScript.

## ✨ Особенности

- **Точный подсчет времени:** Отображает оставшееся время в месяцах, днях, часах, минутах и секундах.
- **Грамматически верный вывод:** Реализована функция склонения слов на русском языке (например: *1 день, 2 дня, 5 дней*).
- **Автоматическое обновление:** Таймер сам переключается на следующий год, когда текущий Новый год наступает.
- **Адаптивный дизайн:** Интерфейс корректно отображается как на десктопах, так и на мобильных устройствах.
- **Современный UI:** Использован эффект *Glassmorphism* (матовое стекло) и фоновая анимация падающего снега.

## 🛠️ Технологии

Проект написан на чистом (ванильном) стеке без использования сторонних библиотек и фреймворков:
- **HTML5** — семантическая разметка.
- **CSS3** — стилизация, Flexbox, CSS-анимации, медиа-запросы.
- **JavaScript (ES6+)** — логика таймера, работа с объектом `Date`, манипуляции с DOM, использование `setInterval`.
